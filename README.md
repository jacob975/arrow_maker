# 介紹
建安十三年，曹老闆舉兵八十萬攻打荊州，與孫劉聯軍會獵於赤壁，周瑜要求諸葛亮要在十天之內造出十萬隻箭矢。諸葛亮一怒之下怒了一下，言道：「十天只怕會延誤戰機，我一天就能造出十萬隻箭矢。」
# 遊戲規則
點擊畫面即可製造箭矢，每次點擊會製造一隻箭矢。
畫面中間上方會顯示目前製造的箭矢數量，當製造的箭矢數量達到十萬隻時，寫下達成時間，但遊戲繼續。
遊戲提供以下方法製造箭矢
1. 點擊畫面，預設一次一隻箭矢。每次升級消耗200x等級的箭矢，升級後點擊一次會多製造一隻箭矢，可無限升級。
2. 用1k隻箭矢可以解鎖苦力趙雲，解鎖後會自動製造箭矢，預設每秒結算一次，每次製造10隻箭矢。每次升級消耗`200x(等級^1.4)`的箭矢，升級後每秒會多製造10隻箭矢，可無限升級。
3. 用5k隻箭矢可以解鎖濫好人魯肅，解鎖後會自動製造箭矢，預設每 3 秒結算一次，每次 +150。每次升級消耗`5kx(等級^1.4)`的箭矢，升級後每次會多製造150隻箭矢，可無限升級。
4. 用20k隻箭矢可以解鎖曹老闆的草船，解鎖後使用者可以每10秒使用一次草船，固定消耗1000隻箭矢，每次使用後獲得隨機整數，範圍 1000 到 3000（含端點）。每次升級消耗`20kx(等級^1.4)`的箭矢，升級後每次使用草船的箭矢亂數下限增加500、上限增加2000，可無限升級。
# 里程碑與結束條件
無勝利條件，遊戲繼續進行直到玩家不想玩為止。
顯示首次達成持有十萬隻箭矢的時間。
# 備註
等級從1開始，升級後等級會增加1。
升級費用使用當前等級計算。
使用四捨五入到整數的方式計算升級費用。
