# 介紹
建安十三年，曹老闆舉兵八十萬攻打荊州，與孫劉聯軍會獵於赤壁，周瑜要求諸葛亮要在十天之內造出十萬隻箭矢。諸葛亮一怒之下怒了一下，言道：「十天只怕會延誤戰機，我一天就能造出十萬隻箭矢。」
# 遊戲規則
點擊畫面即可製造箭矢，每次點擊會製造一隻箭矢。
畫面中間上方會顯示目前製造的箭矢數量，當製造的箭矢數量達到十萬隻時，寫下達成時間，但遊戲繼續。
遊戲提供以下方法製造箭矢
1. 點擊畫面，預設一次一隻箭矢。每次升級消耗200x等級的箭矢，升級後點擊一次會多製造一隻箭矢，可無限升級。
2. 趙雲
    + 描述：子龍將軍忠義無雙，手搓箭矢的速度也是一等一的。
    + 持有1k隻箭矢時解鎖，解鎖當下不可用。
    + 解鎖後會自動製造箭矢，預設每秒結算一次，每次製造10隻箭矢。
    + 每次升級消耗`200x(等級^1.4)`的箭矢，升級後每秒會多製造10隻箭矢，可無限升級。
3. 魯肅：
    + 描述：魯肅從孫吳那裡搖人來了。
    + 持有5k隻箭矢時解鎖，解鎖當下不可用。
    + 解鎖後會自動製造箭矢，預設每 3 秒結算一次，每次 +150。
    + 每次升級消耗`1kx(等級^1.4)`的箭矢，升級後每次會多製造150隻箭矢，可無限升級。
4. 曹老闆草船
    + 描述：我只說造箭矢，沒說誰造的箭矢。
    + 持有20k隻箭矢時解鎖，解鎖當下不可用。
    + 解鎖後使用者可以每10秒使用一次草船，每次使用後獲得隨機整數，範圍 1000 到 3000（含端點）。
    + 每次升級消耗`5kx(等級^1.4)`的箭矢，升級後每次使用草船的箭矢亂數下限增加500、上限增加2000，可無限升級。
5. 借東風
    + 描述：東風一吹，箭矢就來了。
    + 持有100k隻箭矢時解鎖，解鎖當下不可用。
    + 畫面上會顯示一個指南針，指南針的方向會隨機改變。箭矢製造的速度正比於指南針指向的方向，指向右邊時速度最快，指向左邊時速度最慢。最快的製造速度為每秒製造2000隻箭矢，最慢的製造速度為每秒製造500隻箭矢。
    + 每次升級消耗`20kx(等級^1.4)`的箭矢，升級後每次指南針指向右邊時製造速度增加2000隻、指向左邊時速度增加500隻，可無限升級。
6. 奇門遁甲
    + 子不語怪力亂神，除非。
    + 持有500k隻箭矢時解鎖，解鎖當下不可用。
    + 在畫面上顯示一個九宮格如下

    |生|休|開|
    |---|---|---|
    |傷|中軍|驚|
    |杜|景|死|

        在九宮格下方顯示一個長度為四的字串，從「生休開傷驚杜景死」中隨機選擇四個並隨機排列，可重複選擇。如果使用者以正確順序連續點九宮格中相對應的文字，則獲得箭矢X隻，並重置該字串。體感上4秒可以使用一次。
    + 等級1時X=20k，每提升一級，X增加20k。
    + 每次升級消耗`100kx(等級^1.4)`的箭矢，可無限升級。
7. 天壇
    + 描述：科學的盡頭。
    + 持有2M隻箭矢時解鎖，解鎖當下不可用。
    + 生成一個正方形，內部有一個移動速度為1pix每秒的星星。使用者可以用滑鼠點擊拖曳移動一個由虛線圍成的圓形。如果星星在圓形內部，則使用者每秒都能獲得箭矢X隻。
    + 等級1時X=20k，每提升一級，X增加20k。
    + 每次升級消耗`500kx(等級^1.4)`的箭矢，可無限升級。
# 里程碑與結束條件
初始顯示兩個製箭方法，分別是點擊畫面和趙雲。
解鎖一個製箭方法才能顯示下一個製箭方法。
無勝利條件，遊戲繼續進行直到玩家不想玩為止。
顯示首次達成持有十萬隻箭矢的時間。
# 備註
等級從1開始，升級後等級會增加1。
升級費用使用當前等級計算。
使用四捨五入到整數的方式計算升級費用。
# 金手指
在瀏覽器的開發者工具中輸入以下指令可以獲得大量箭矢：
```javascript
localStorage.setItem('arrows', '1000000000');
```
